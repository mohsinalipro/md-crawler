# Antigravity SDK Documentation

Welcome to the official documentation for the **Antigravity SDK**—a powerful toolkit for building advanced agentic workflows, multi-agent systems, and self-correcting programs.

Here you will find everything you need to build, test, and deploy agents that can perform complex reasoning tasks and interact with external APIs safely.

## 💡 Key Design Philosophies

* **Agnostic Integrations**: Supports local and hosted LLMs out of the box.
* **Resilient Workflows**: Built-in auto-retry, error fallback, and sandboxed runtimes.
* **Extensible Architecture**: Easily load markdown-based rules and custom skills.
